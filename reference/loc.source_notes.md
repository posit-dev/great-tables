## loc.source_notes


Target the source notes.


Usage

``` python
loc.source_notes()
```


With [loc.source_notes()](loc.source_notes.md#great_tables.loc.source_notes), we can target the source notes in the table. This is useful when applying custom with the <a href="GT.tab_style.html#great_tables.GT.tab_style" class="gdls-link"><code>tab_style()</code></a> method. That method has a `locations=` argument and this class should be used there to perform the targeting. The 'source_notes' location is generated by <a href="GT.tab_source_note.html#great_tables.GT.tab_source_note" class="gdls-link"><code>tab_source_note()</code></a>.


## Returns


`LocSourceNotes`  
A [LocSourceNotes](loc.source_notes.md#great_tables.loc.source_notes) object, which is used for a `locations=` argument if specifying the source notes.


## Examples

Let's use a subset of the [gtcars](data.gtcars.md#great_tables.data.gtcars) dataset in a new table. Add a source note (with <a href="GT.tab_source_note.html#great_tables.GT.tab_source_note" class="gdls-link"><code>tab_source_note()</code></a> and style the source notes section inside <a href="GT.tab_style.html#great_tables.GT.tab_style" class="gdls-link"><code>tab_style()</code></a> with `locations=loc.source_notes()`.


``` python
from great_tables import GT, style, loc
from great_tables.data import gtcars

(
    GT(gtcars[["mfr", "model", "msrp"]].head(5))
    .tab_source_note(source_note="From edmunds.com")
    .tab_style(
        style=style.text(color="blue", size="small", weight="bold"),
        locations=loc.source_notes()
    )
)
```
