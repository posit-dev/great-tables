## loc.footer


Target the table footer.


Usage

``` python
loc.footer()
```


With [loc.footer()](loc.footer.md#great_tables.loc.footer) we can target the table's footer, which currently contains the source notes (and may contain a 'footnotes' location in the future). This is useful when applying custom styling with the <a href="GT.tab_style.html#great_tables.GT.tab_style" class="gdls-link"><code>tab_style()</code></a> method. That method has a `locations=` argument and this class should be used there to perform the targeting. The 'footer' location is generated by <a href="GT.tab_source_note.html#great_tables.GT.tab_source_note" class="gdls-link"><code>tab_source_note()</code></a>.


## Returns


`LocFooter`  
A [LocFooter](loc.footer.md#great_tables.loc.footer) object, which is used for a `locations=` argument if specifying the footer of the table.


## Examples

Let's use a subset of the [gtcars](data.gtcars.md#great_tables.data.gtcars) dataset in a new table. Add a source note (with <a href="GT.tab_source_note.html#great_tables.GT.tab_source_note" class="gdls-link"><code>tab_source_note()</code></a> and style this footer section inside of <a href="GT.tab_style.html#great_tables.GT.tab_style" class="gdls-link"><code>tab_style()</code></a> with `locations=loc.footer()`.


``` python
from great_tables import GT, style, loc
from great_tables.data import gtcars

(
    GT(gtcars[["mfr", "model", "msrp"]].head(5))
    .tab_source_note(source_note="From edmunds.com")
    .tab_style(
        style=style.text(color="blue", size="small", weight="bold"),
        locations=loc.footer()
    )
)
```
