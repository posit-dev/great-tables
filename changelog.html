v0.21.0: add cols_merge(), cols_label_with(), opt_css()

2026-03-03 · GitHub

Deprecations

  • The opt_all_caps() method now uses loc.column_labels, loc.stub, and loc.row_groups as arguments to locations=; using strings to represent the locations still works but triggers a deprecation warning (https://github.com/posit-dev/great-tables/pull/436)

Features

  • Added the cols_merge() method for combining cell contents across two or more columns by @rich-iannone in https://github.com/posit-dev/great-tables/pull/780
  • Added the cols_label_with() method for function-based column relabeling by @machow in https://github.com/posit-dev/great-tables/pull/816
  • Added the fmt_engineering() method by @rich-iannone in https://github.com/posit-dev/great-tables/pull/786
  • The new opt_css() method lets you define arbitrary CSS rules for an HTML table by @rich-iannone in https://github.com/posit-dev/great-tables/pull/775
  • Added the ability to define container padding in tab_options() by @thriller08 in https://github.com/posit-dev/great-tables/pull/802

Fixes

  • Refactored internal auto-align code for compatibility with Pandas 3.0.0 by @machow in https://github.com/posit-dev/great-tables/pull/810
  • Added int to the RowSelectExpr definition by @tylerriccio33 in https://github.com/posit-dev/great-tables/pull/800
  • Updated the Polars selector type annotation to use Selector by @schmidma in https://github.com/posit-dev/great-tables/pull/770
  • Annotated the pipe method’s first argument type as GT by @FBruzzesi in https://github.com/posit-dev/great-tables/pull/757
  • We now correctly handle the interactive_data_values= properly in Nanoplots by @lorenzo-w in https://github.com/posit-dev/great-tables/pull/792
  • Group labels are now modified when render_formats() is called by @juleswg23 in https://github.com/posit-dev/great-tables/pull/769
  • The opt_all_caps() method has been refactored by @jrycw in https://github.com/posit-dev/great-tables/pull/436

Docs

  • Docs for the val_fmt_engineering() were added to the project website docs by @jrycw in https://github.com/posit-dev/great-tables/pull/808
  • Corrected the fns= parameter docs in the fmt() method by @rich-iannone in https://github.com/posit-dev/great-tables/pull/818
  • Improved the docs for the from_column() function by @jrycw in https://github.com/posit-dev/great-tables/pull/766
  • Added a gt-extras example to the project website’s Examples page by @juleswg23 in https://github.com/posit-dev/great-tables/pull/759
  • Added an example demonstrating usage of the define_units() function by @jrycw in https://github.com/posit-dev/great-tables/pull/446

Chores

  • Updated the copyright end year to 2026 by @rich-iannone in https://github.com/posit-dev/great-tables/pull/804
  • Several typos were fixed by @kianmeng in https://github.com/posit-dev/great-tables/pull/798
  • Added Plausible analytics to the project website by @rich-iannone in https://github.com/posit-dev/great-tables/pull/806
  • Added several tests for the opt_all_caps() method by @rich-iannone in https://github.com/posit-dev/great-tables/pull/812
